StateMachineService
service in module fsm

Description

A Service for a Finite State Machine
Wrapper for javascript-state-machine

Methods

  • create()

    Sets up the FSM States

    1. initial -> makeReady()
    2. ready -> play()
    3. playing -> stop()
    4. paused ( show best word ) -> finish()
    5. paused ( show best word && no tiles ) -> end()
    6. done -> reset()
    7. resetting -> initialise()
